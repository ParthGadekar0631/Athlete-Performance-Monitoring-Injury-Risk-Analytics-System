# GitHub Pages Dashboard

This folder contains the static, GitHub Pages-ready version of the athlete monitoring dashboard.

The dashboard reads `dashboard_export.csv` directly in the browser, so it does not require FastAPI or PostgreSQL when hosted on GitHub Pages.
